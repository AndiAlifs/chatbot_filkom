# Chatbot Enigma

Enigma chatbot is a machine learning based chatbot that can respond user input message to a proper responds, such as greeting, asking name, etc.
## Requirement
* Python
* Numpy
* Tensorflow
* NLTK


## Usage
1. Clone the repository
2. Run by command
```bash
python chatbotMLbased.py
```
